Festivent

Look up events from multiple famous sources, in one unique android application

View on GitHub Download .zip Download .tar.gz

Festivent is a latest android design application, featuring a clean User-interface to view local events in a list or on Google map

Features

  • Search events based on selected location or current location
  • Display events in neat card view with basic event information up front
  • See events pin-pointed on map with basic event information
  • Choose search radius
  • Choose desired event date
  • Save events for easy access
  • Get google direction for events with one click
  • Mark your Calendar for selected events
  • Share events with friend and family
